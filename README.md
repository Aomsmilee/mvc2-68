# mvc2-68

**เลือกทำข้อที่ 1**

โปรแกรมนี้พัฒนาโดยใช้ภาษา Python
ใช้สถาปัตยกรรม MVC (Model-View-Controller)

## Structure
- Model: จัดการข้อมูลข่าวลือ รายงาน และการตรวจสอบ
- View: GUI (Tkinter)
- Controller: ควบคุมการทำงาน

## How to run
```bash
python3 init_db.py
python3 main.py
